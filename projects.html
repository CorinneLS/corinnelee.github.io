Mysh (my shell)

Software tools and system programming | C, file descriptors, socket programming

- Wrote a C program to implement a fully functional shell called mysh. mysh provides a vast range of functionality matching, or even outperforming, that of other builtin shells in many ways.
- mysh supports the following commands as builtins: echo, ls, cd, cat, wc, pipes(|), bg (&), kill, ps, exit, start-server, close-server, send, start-client
- Implemented pipes, forks and signals to start and kill process in the background
- Implemented socket programming to communicate over a network: initiated a background server on the provided port number. The non-blocking server allows multiple connections at the same time. A client connected to the server can send multiple messages.
